# Add your artwork files here

This folder is where you'll add your artwork images!

## How to Add Artwork:

1. **Add your image files** to this folder
   - Supported formats: JPG, PNG, GIF, WebP
   - Recommended size: 800-1200px for web optimization

2. **Update the artworks.js file** (`src/artworks.js`)
   - Add an entry for each artwork with title, description, category, and file path
