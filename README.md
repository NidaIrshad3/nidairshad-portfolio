## ğŸ“ Adding Artwork

1. Add your image files to `public/artworks/`
2. Edit `src/artworks.js` and add entries for each artwork:

```javascript
{
  id: 1,
  title: "Your Artwork Title",
  description: "Brief description",
  category: "sketch", // or: pixel, doodle, digital, 3d
  image: "/artworks/your-image.jpg",
  featured: false
}
```

3. Save and your artwork appears on the site!
